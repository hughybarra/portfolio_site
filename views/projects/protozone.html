Protozone

Protozone is an action script based game built on top of the starling api. Our team was tasked with building a game for a group project. My team and I managed to put this project together in just a few days

My role in this project

  • Enemy Movement
  • Enemy AI
  • Collision Detection
  • Particle Effects Design / Implementation
  • Bullet management
  • Level Manager
  • Memory allocation and optimization

This is by far one of the most fun projects I have ever worked on
