Weather API

The Weather API is a simple api to get the weather forcast for your zip code location. The site uses the yahoo weather sevice and uses a zip code to pull down data from their server. Once the data is on my server I parse it and display in the browser.

While interviewing with a company in Orlando Floria I was given this site as a test project. The purpose of this site was to prove that I konw how to do the basics of LAMP, server develpment, and javascript

The Technology stack

  • LAMP
  • Javascript
  • CSS3
  • JQuery
